Introduction

Qu'est-ce que Cat Classifier ?

Cat Classifier est une application de machine learning qui utilise la régression logistique pour classifier les images comme contenant un chat ou non. Ce projet démontre l'implémentation d'un réseau de neurones simple pour la classification d'images.

🎯 Objectifs pédagogiques
  • Fondamentaux de la régression logistique
  • Réseau de neurones à une couche
  • Création d'API web avec Flask
  • Déploiement de modèles ML
🔬 Technologies
Python Flask NumPy PIL Bootstrap 5

Fondements Mathématiques

Le modèle utilise une régression logistique structurée comme un neurone artificiel :

Forward Propagation:
z = wᵀ x + b
a = σ(z) = 1 / (1 + e⁻ᶻ)
Fonction de Coût (Log Loss):
J = -1/m Σ [y⁽ⁱ⁾ log(a⁽ⁱ⁾) + (1-y⁽ⁱ⁾) log(1-a⁽ⁱ⁾)]

Installation

1. Cloner le projet
git clone https://github.com/leadylearn/logistic-regression-neural-network.git
cd logistic-regression-neural-network
2. Environnement virtuel
python -m venv venv
# Activation (Linux/Mac)
source venv/bin/activate
# Activation (Windows)
venv\Scripts\activate
3. Dépendances & Lancement
pip install flask numpy matplotlib pillow scipy h5py
python app.py

L'application sera disponible sur http://localhost:5000

Exemples

Appel avec Python (Requests)
import requests

url = "http://localhost:5000/predict"
files = {'image': open('mon_chat.jpg', 'rb')}
response = requests.post(url, files=files)

print(response.json())
